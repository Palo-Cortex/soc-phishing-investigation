# Completing the SOC Phishing Investigation & Response XSIAM Tenant Configuration


### 1. Add Recommended Playbook Triggers

The Phishing Investigation and Response playbooks are triggered based on playbook triggers. When you are confident in 
the functionality of the playbook, you will need to add the recommended triggers to your Playbook Trigger configuration.

1. In XSIAM, navigate to **Incident Response > Incident Configuration > Playbook Triggers**
2. Click the **View Recommendation** button in the top right of the screen 
3. Search the table for the following playbook triggers, based on the trigger name:
   1. SOC Phishing
4. Click **Add selected triggers**
